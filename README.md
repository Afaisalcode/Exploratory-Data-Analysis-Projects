# Exploratory Data Analysis (EDA) Projects

This repository contains Exploratory Data Analysis (EDA) on four different real-world datasets as part of the PW Skills course.

---

## 📂 Projects Overview

### 1. 🏍️ Bike Details Analysis
* **Description:** Analysis of 1,061 used bike listings to understand resale prices, kilometers driven, and ownership trends.
* **Key Focus:** Outlier detection (IQR), price depreciation, and bivariate analysis between price and manufacturing year.

### 2. 🚗 Car Sales Analysis
* **Description:** Dataset capturing second-hand car transactions, customer demographics, and dealer performance.
* **Key Focus:** Pricing distribution across body styles, region-wise sales analysis, and correlation with customer income.

### 3. 🛒 Amazon Sales & Product Analysis
* **Description:** 1K+ Amazon product listings featuring prices, discount percentages, customer ratings, and reviews.
* **Key Focus:** Rating vs. discount correlation, top category performance, and keyword extraction from reviews.

### 4. 🎵 Spotify Popular Hip-Hop Music
* **Description:** Collection of ~500 popular hip-hop tracks and artists on Spotify.
* **Key Focus:** Track popularity distributions, track duration analysis, and top artist performance using scatter plots, box plots, and swarm plots.

---

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
